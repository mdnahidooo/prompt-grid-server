# 🚀 PromptGrid – AI Prompt Sharing & Marketplace Platform

PromptGrid is a modern full-stack marketplace where users can create, discover, share, bookmark, review, and monetize AI prompts for popular AI tools such as ChatGPT, Gemini, Claude, Midjourney, and more.

The platform provides a community-driven ecosystem where prompt creators can publish high-quality prompts, build an audience, track performance analytics, and offer premium content. Users can explore trending prompts, bookmark favorites, leave reviews, and unlock premium prompts through secure Stripe payments.

---

## 🌐 Live URL

## 🎯 Project Goal

The goal of **PromptGrid** is to build a secure and scalable AI prompt marketplace where creators and users can collaborate, share knowledge, and improve productivity through high-quality AI prompts. The platform empowers creators to showcase their work while allowing users to discover, save, review, and access both free and premium prompt collections.
